# Automatic Operation
## Basic Principle
A train runs always on a route. A route starts and ends on a track. The destination track has to be configured with at least one feedback.

Using signals as start or target in routes is obsolete.

## Quick Guide
1. Create a locomotive (../menu->Locomotives)
2. Create starting track (With right mouse click on layout->add track)
3. Create feedback for destination track (With right mouse click on layout->add feedback) (This can also be a non visible feedback)
4. Create destination track (With right mouse click on layout->add track)
5. Configure feedback to destination track (With right mouse click on layout->edit track->feedbacks)
6. Create route (With right mouse click on layout->add route) (This can also be a non visible route)
7. Configure route with start and destination track and a stop feedback (With right mouse click on route->edit track->automatic)
8. If needed configure other elements to the route (With right mouse click on route->edit route->relations)
9. Put locomotive onto track (physically on the layout)
10. Set locomotive (With left mouse click on track->set locomotive)
11. Start locomotive (With left mouse click on track->start locomotive)
