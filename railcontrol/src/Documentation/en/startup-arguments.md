# Startup arguments
RailControl can be started from Windows-Explorer, from a Linux-Explorer or from the command line terminal.

If RailControl is started from the command line there are some options available:
    --config=ConfigFile
Read config file with file name ConfigFile (default ConfigFile: railcontrol.conf)

     -d    --daemonize
Daemonize RailControl. Implies -s.

    --logfile=LogFile
Write a logfile to file LogFile (default LogFile: railcontrol.log). If no logfile is required LogFile has to be empty.

    -h    --help
Show help with argument descriptions

    -s    --silent
Omit writing to console
