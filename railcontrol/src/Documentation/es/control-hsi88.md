## Configuración de HSI-88 RS232

RailControl solamente está soportando el HSI-88 RS232. La version USB no está soportado.

Para configurar HSI-88 se tiene que seleccionar el puerto serial.

Especialmente bajo Linux se tiene que considerar la pagina [Convertidor de RS-232 a USB](#convertidor-de-rs-232-a-usb).

Además se tiene que configurar el numero de módulos de retroseñales con 8 conexiones que están conectado a HSI-88. El maximo son 62 módulos con 8 conexiones. Módulos con 16 conexiones vale dos módulos.
