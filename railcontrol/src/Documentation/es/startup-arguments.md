# Argumentos de inicio
RailControl puede ser iniciado de Windows-Explorer, de un Linux-Explorer o de la línea de comando.

Si RailControl está iniciado de la línea de comando hay algunas opciones disponibles:

    --config=ConfigFile
Lee el archivo de configuración con el nombre ConfigFile (ConfigFile pro defecto: railcontrol.conf)

    -d    --daemonize
Daemonize RailControl. Implica -s

    --logfile=LogFile
Escibe un archivo de registro con el nombre LogFile (LogFile por defecto: railcontrol.log). Si no se requiere un archivo de registro LogFile tiene que ser vacio.

    -h    --help
Muestra una ayuda

    -s    --silent
Omite escribir en la consola
