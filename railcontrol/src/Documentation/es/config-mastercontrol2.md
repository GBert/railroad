## Configuración de Tams MasterControl 2

Para configurar Tams MasterControl 2 se tiene que seleccionar la dirección IP del dispositivo. Para configurar una dirección IP se tiene que consultar los instrucciones de uso del MasterControl 2.

Además se tiene que configurar el numero de módulos de retroseñales con 8 conexiones que están conectado a MasterControl. El maximo son 104 módulos con 8 conexiones. Módulos con 16 conexiones vale dos módulos.

Porque los principios del protocolo este control no puede procesar retroseñales muy cortas. Por esto está recomendado usar otras posibiliidades de retroseñales para la operación automatica.
